### 方式二：安装 BlueStacks 模拟器（推荐用于真实体验）

**在电脑上运行真实的 Android 应用！**

#### 步骤 1：安装 BlueStacks
1. 访问：https://www.bluestacks.com/
2. 下载并安装 BlueStacks 5
3. 安装完成后启动 BlueStacks

#### 步骤 2：获取预构建的 APK
由于当前环境无法构建，您可以：
- 使用其他有 Android Studio 的电脑构建 APK
- 或者使用在线构建服务（如 GitHub Actions）

#### 步骤 3：安装 APK
1. 双击 APK 文件
2. 或拖拽 APK 到 BlueStacks 窗口
3. 等待安装完成
4. 在 BlueStacks 中点击应用图标运行

**优点**：
- 真实的 Android 应用体验
- 支持所有功能（包括通知）
- 可以测试完整的交互

---

### 方式三：配置完整开发环境（高级用户）

**如果您想自己构建和修改代码**

#### 前置要求
1. **安装 Android Studio**
   - 下载：https://developer.android.com/studio
   - 安装 Android Studio 会自动安装所有必需的工具

2. **或安装 Android SDK 命令行工具**
   - 下载：https://developer.android.com/studio#command-tools
   - 配置环境变量
   - 安装必要的 SDK 组件

#### 构建步骤
```powershell
# 1. 打开项目
# 在 Android Studio 中打开项目文件夹

# 2. 等待 Gradle 同步完成

# 3. 连接设备或启动模拟器

# 4. 点击 Run 按钮运行
```

**优点**：
- 完整的开发环境
- 可以修改和调试代码
- 支持热重载
